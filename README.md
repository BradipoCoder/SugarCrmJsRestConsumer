# SugarCrm JS Rest Consumer

A promise based javascript rest client for SugarCRM Rest API v.4.1.
 
## Features/Implemented methods
- login
- logout
- seamless_login
- get_user_id
- get_server_info
- get_available_modules
- get_module_fields
- get_entry_list
- get_entries_count
- get_entries
- get_entry
- set_entries
- set_entry
- raw post function to send any method and data to Rest API
